Getting Started
⬛
Welcome to MCDesign v0.2
A powerful, browser-based pixel editor for Minecraft texture packs. No downloads. No installations.
What's new in v0.2:
HSV + RGB color pickers · Rectangle & Ellipse tools · Move tool · Layers system · Hue/Saturation & Brightness/Contrast adjustments · Rotate · Invert/Grayscale · Improved light mode
What you'll learn:
① How to import a texture pack
② How to use the drawing tools
③ How to export your finished pack
📦
Importing a Texture Pack
Load any resource pack ZIP or Minecraft JAR to start editing.
Option A — Import ZIP or JAR:
Click File → Import Texture Pack or drag-and-drop a ZIP/JAR onto the drop zone in the sidebar.
Option B — New Texture:
Use File → New Texture… to create a blank canvas. 16×16 is the standard Minecraft block texture size.

Color Picker
Three ways to pick your color.
HSV Picker:
Click and drag the gradient square to pick hue, saturation and value. Use the hue bar to change hue, and the alpha bar for transparency.
RGB Sliders:
Use the R, G, B sliders for precise numeric control. The A slider controls alpha (opacity).
HEX Input:
Type a hex code directly (e.g. #7dd4f0). Supports 6 or 8 digit hex (with alpha).
📤
Exporting Your Pack
Export a complete resource pack ZIP ready to drop into Minecraft.
Export as ZIP:
Use File → Export Pack as ZIP. MCDesign will generate a pack.mcmeta and build the correct folder structure automatically.

For Minecraft 1.21.9+, MCDesign automatically includes min_format and max_format pack format fields as required.
Individual textures:
Use Ctrl+S to save the current texture in memory, or Ctrl+E to export as a standalone PNG file.
