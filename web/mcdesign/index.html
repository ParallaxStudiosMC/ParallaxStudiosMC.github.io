Getting Started
⬛
Welcome to MCDesign
A simple, fast, and efficient browser-based pixel editor for Minecraft that requires ZERO downloads or installations, and is very powerful.
How do I get started?
To get started with MCDesign, simply import a JAR file, or ZIP file of the game's assets into MCDesign, and it will handle importing them all for you, just sit back while MCDesign imports them! After importing, MCDesign will sort the textures into sections, making organization easier. You can now create your textures with ease! Ready to export? MCDesign will help you there too! Just select the game version the pack is for, and customize its pack.mcmeta file, and export!
What you'll learn:
① How to import a texture pack
② How to use the drawing tools
③ How to export your finished pack
📦
Importing a Texture Pack
Load any resource pack ZIP to start editing.
Option A — Import a JAR file or ZIP file:
Click File → Import Texture Pack (ZIP / JAR) or drag-and-drop a ZIP, or JAR onto the drop zone.
Option B — New Texture:
Use File → New Texture… to create a blank canvas. 16×16 is standard for most Minecraft blocks.
🖥
The Interface
Three panels, one canvas.
Left Panel — Texture Browser:
Browse all textures organized by category. Click any texture to open it.
Center — Canvas:
Your editing area. Scroll or use +/- to zoom. Press F to fit to screen.
Right Panel — Color & Settings:
Pick colors, adjust brush, and use the Minecraft palette.

Exporting Your Pack
Export a complete resource pack ZIP ready to drop into Minecraft.
Export as ZIP:
Use File → Export Pack as ZIP. MCDesign will generate a pack.mcmeta file for you.
Save individual textures:
Use Ctrl+S to save the current texture, or Ctrl+E to export as a standalone PNG.
